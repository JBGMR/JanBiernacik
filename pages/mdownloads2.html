Windows 10 APPX remover

Removes every Windows Store/UWP package (exept Edge, Defender and Settings) and cleans up your system with one command. It also removes the store itself. Make sure to run this after every major Windows update.

Telemetry disabler

Disabled Windows telemetry via a registry entry. this means that Microsoft doesn't collect any diagnostics data about your device. This can protect your privacy a bit more.

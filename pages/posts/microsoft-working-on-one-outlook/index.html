Microsoft working on a replacement for the Mail and Calendar apps in Windows 10

Microsoft is planning to replace the default Mail and Calendar apps in Windows 10 with project "Monarch" also called One Outlook (which already got mentioned last year). So far there is not much information besides some screenshots and that it will be an Outlook client for all platforms including the Web.

The client will not only replace the classic Mail and Calendar apps on Windows 10 but also the Win32 version of Outlook sometime next year.

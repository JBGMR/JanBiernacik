Tutorials

Telemetry

Do you want Microsoft to collect your personal data? I'm sure the answer is no. Here is a step by step guide on how to disable telemetry:

Step 1: Open the Windows-Run dialog and type in regedit.exe.

Step 2: Navigate to 'Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DataCollection'.

Step 3: Create a new DWORD (32 bit) entry and name it 'AllowTelemetry' (Skip this step if it already exists) and set its value to 0.

Done.

Web search

If you are using Windows 10 version 2004 or newer, you may have noticed that the old way to disable web search no longer works. Thankfully, there is a new way.

Step 1: Open the Windows-Run dialog and type in regedit.exe.

Step 2: Navigate to 'Computer\HKEY_CURRENT_USER\SOFTWARE\Policies\Microsoft\Windows\Explorer'.

Step 3: Create a new DWORD (32 bit) entry and name it 'DisableSearchBoxSuggestions' and set its value to 1.

Step 4: Restart explorer, log out or reboot.

Startup Apps

Unnecessary startup applications and programs may have an impact on performance and boot time. To check for them follow these steps:

Step 1: Open task manager.

Step 2: Navigate to the Startup tab.

Step 3: Select the unnecessary Startup program and hit disable.
